Carpeta de videos de clases (.mp4)

- Coloca aquí tus archivos de video en formato .mp4.
- Nombres sugeridos: intro.mp4, clase1.mp4, clase2.mp4, ..., clase7.mp4, cierre.mp4
- Enlaza desde src/data/modules.ts con rutas absolutas: /videos/intro.mp4, /videos/clase1.mp4, etc.
- Mantén los archivos por debajo de ~100MB si vas a subir a GitHub. Para archivos más grandes, considera almacenar fuera del repo o usar Git LFS.
